Gmail setup instructions

  1. Update the employee details above.
  2. Click Copy Signature.
  3. Open Gmail → Settings → See all settings → Signature.
  4. Create or select a signature, then paste.
  5. Scroll to the bottom of Gmail settings and click Save Changes.
